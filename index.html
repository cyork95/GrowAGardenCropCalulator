Events and Crop Mutations Explanation

Events in Grow a Garden are special occurrences that can trigger unique mutations on crops, increasing their value. Some mutations are tied to weather events, while others come from seasonal or limited-time events. Below is a summary of all events and their effects on crop mutations, including multipliers and compatibility. Mutations from events can stack with others unless specified.

Event Mutation(s) Multiplier(s) How It Affects Mutations Compatibility/Notes
Windy Event Windstruck 2x Occurs rarely; affects crops with wind, granting Windstruck mutation to increase value. Compatible with most; removed by Tempestuous. No replacement rules.
Tornado Event Twisted 5x Triggered by severe wind; crops gain Twisted mutation during the event. Compatible with most; removed by Tempestuous. No specific conditions.
Rain Event Wet, Drenched 2x (Wet), 5x (Drenched) Hydrates crops automatically; can grant Wet or Drenched mutations based on intensity. Wet/Drenched replaced by Frozen or Clay. Compatible with others.
Storm Event Shocked 100x Increases mutation rates; crops may gain Shocked mutation during storms. Compatible with most; no replacement rules noted.
Snow Event Chilled, Frozen 2x (Chilled), 10x (Frozen) Affects crop temperature; grants Chilled or Frozen mutations. Chilled/Frozen removed by Frozen (upgrades). Compatible with others.
Heat Wave Event Sundried 85x Increases heat; crops can gain Sundried mutation if exposed. Compatible with most; removed by Paradisal.
Corrupted Event Static, Corrupt, CorruptChakra, CorruptFoxfireChakra 8x (Static), 20x (Corrupt), 15x (CorruptChakra), 90x (CorruptFoxfireChakra) Uses Raiju fox every 10.30 minutes to apply Static; other corrupted mutations during event. CorruptChakra removed by HarmonisedChakra. Compatible with most.
Chakra Event Chakra, HarmonisedChakra, FoxfireChakra, HarmonisedFoxfireChakra 15x (Chakra), 35x (HarmonisedChakra), 90x (FoxfireChakra), 190x (HarmonisedFoxfireChakra) Introduces Chakra-based mutations; higher tiers combine lower ones. Harmonised removes lower Chakra variants. Only one per crop.
Spaghetti Event Sauce, Meatball, Pasta, Spaghetti 3x (Sauce/Meatball), 4x (Pasta), 15x (Spaghetti) Event for food-themed mutations; Spaghetti combines the others. Spaghetti removes Sauce/Meatball/Pasta. Compatible with others.
Amber Event Amber, OldAmber, AncientAmber 10x (Amber), 20x (OldAmber), 50x (AncientAmber) Progressive Amber stages during event; upgrades over time. Only one Amber stage per crop; cannot stack stages.
Zombified Event (Halloween) Zombified 25x Halloween event grants Zombified mutation to crops. Compatible with most; no replacement rules.
Disco Event Disco 125x Event for Disco mutation, adding visual effects and value. Compatible with most; no replacement rules.
Blood Moon Event Bloodlit 5x Night event for Bloodlit mutation. Compatible with most; no replacement rules.
Moonlit Event Moonlit 2x Night event for Moonlit mutation. Compatible with most; no replacement rules.
